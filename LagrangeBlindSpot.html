深空边缘：纪元 2240

“就是因为我们行事过于鲁莽，过于相信大炮和传感器，我们犯了错，正为此付出代价。这是我们的错，也仅仅是我们的错。——［波兰］斯坦尼斯瓦夫·莱姆”

快速了解世界观

时间：公元2240年

在人类走出太阳系后的第一个世纪，文明并没有迎来真正的黄金时代。

半个世纪前，一种被称为C-62（“黑血”） 的高能同位素矿物被发现，它能够提供近乎无限的清洁能源，让人类文明得以飞速发展并继续向外扩张，也让整个星系陷入漫长的资源战争。

然而，长期接触原矿的人会患上无法治愈的职业病——星尘病，患者会受到不可逆且逐步恶化的神经损伤，器官将逐渐纤维化，同时伴随着严重的幻觉，最终在痛苦中死去。

同数个世纪前的放射性元素镭一样，黑血的危害在发现后的数十年间始终被联盟和企业刻意淡化，直到越来越多矿工死于被称为“星尘病”的慢性疾病，官方的消息封锁才逐渐失效。

但此时一切都已无法挽回，整个文明早已深深依赖上这种能源，内环的城市、星际航道、舰队引擎乃至数十亿人的生活，都建立在黑血之上，没有人不清楚得到它的代价，也没有人能承担失去它的后果。

2240年，战争尚未全面爆发，但和平早已不复存在。

联盟、财团、外环居民、佣兵组织和无数独立势力彼此争斗、合作、背叛，在脆弱的平衡中维持着整个文明的运转；暗杀、暴动、海盗袭击、企业战争和情报渗透几乎每天都在发生，却没有任何一方能够真正摧毁自己的敌人。

因为所有人都明白，他们共同生活在一艘正在缓慢渗水的船上。

在这片广袤而混乱的深空之中，没有谁能真正统治谁。

欢迎来到深空边缘。

🌌 势力分布 // FACTIONS

泛星联盟 (P.S.C)

秩序 / 霸权 / 腐朽

统治地球及内环星域的人类最高政体。

联盟掌握着绝大部分黑血精炼设施、星门航线和军事力量。对内环居民而言，他们是维系文明延续的守护者；而在外环人眼中，他们则是建立在压迫与垄断之上的统治者。

外环（The Frontier）

生存 / 自由 / 混乱

在联盟官方星图上，“外环”指代所有位于边缘航道之外的空间站、矿业殖民地和独立聚居区；而对生活在那里的人而言，这更像是一种身份，而非一个国家。

长期与地球和内环社会隔绝，使外环居民逐渐形成了独特的文化和身份认同，但他们从未真正统一。

这里既有革命者，也有商人；既有理想主义者，也有只想活下去的普通人。

独立防务承包商 (PMCs)

金钱 / 契约 / 硝烟

不参与政治站队的私人武装，游走在联邦与反抗军的夹缝中，他们只服从信用点和合同，并向一切支付报酬的人提供武力。

自治空间站与自由港

贸易 / 中立 / 灰色秩序

遍布深空的独立空间站与自由港。

它们依靠贸易、维修、医疗和情报生意生存，在战争与政治夹缝中维持脆弱的中立。

2190

星历 2190 年：潘多拉的魔盒

探测器“远望者-7”在太阳系外环的柯伊伯带深处，传回了一组异常的辐射读数。随后，第一批勘探队带回了那种散发着幽蓝色微光的黑色矿石——C-62。

它的能量转化率是核聚变的数百倍。一夜之间，地球的能源危机迎刃而解，星际航行从昂贵的奢望变成了日常。人类欢呼雀跃，以为自己终于握住了上帝的权杖，却不知道他们打开的是潘多拉的魔盒。

2205

星历 2205 年：群星的低语

随着开采规模的扩大，第一批长期接触原矿的底层矿工开始出现异常。起初只是轻微的咳嗽和皮肤荧光，随后演变为严重的器官纤维化。

最可怕的是伴随而来的幻觉——患者声称听到了“群星的低语”。泰坦星脉财团试图用高压手段封锁消息，将死者秘密处理。但纸包不住火，这种被称为“星尘病”的绝症最终在整个外环蔓延开来。

2218

星历 2218 年：秩序的铁腕

面对外环日益严重的抗议和黑血走私问题，内环的旧政体意识到，松散的联合已经无法维持对星系的统治。

在泰坦星脉财团的资金支持下，地球及内环各大行星签署了《泛星条约》，正式成立“泛星联盟 (P.S.C)”。联盟迅速组建了第一舰队和战略安全局（SSA），以铁腕手段接管了所有核心星门和精炼设施。

2230

星历 2230 年：鲜血与火花

联盟的压迫终于达到了临界点。外环 12 个主要矿区同时爆发大罢工，要求改善工作环境并提供星尘病的治疗药物。

联盟的回答是第一舰队的轨道轰炸。罢工被血腥镇压，数万人死亡。但在废墟之中，幸存的矿工和逃兵们拿起了武器，“黑火”解放战线正式宣告成立，星系进入了漫长的游击战时代。

2240

星历 2240 年：深空边缘

当前纪元。黑火武装成功劫持了联盟的 7 号采矿驳船，造成大量原矿泄漏。这一事件彻底激怒了联盟最高议会。

随着正规军的集结，各大独立防务承包商（PMC）也闻风而动，纷纷涌入外环寻找发财的机会。在阿刻戎自由港的阴影下，情报贩子们疯狂交换着筹码。全面战争的倒计时，已经开始。
